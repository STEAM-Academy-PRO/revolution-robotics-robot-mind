MCU bootloader
==============

[Documentation root](../index.md)

The bootloader is the code on the MCU that enables firmware updates.

The MCU is a [Microchip ATSAMD51P19A](../assets/SAM-D5x-E5x-Family-Data-Sheet-DS60001507.pdf).
